## Descrição do Projeto

R2D2 Tecnologias é uma landing page desenvolvida em React que apresenta soluções digitais inovadoras para empresas. A aplicação da visibilidade a serviços de tecnologia com design responsivo, componentes reutilizáveis e navegação intuitiva.

### Principais Funcionalidades

- **Header Navegável** - Menu com links âncora para diferentes seções
- **Hero Section** - Banner principal com call-to-action (CTA)
- **Cards de Serviços** - Exibição de 4 serviços digitais com ícones
- **Seção de Diferenciais** - Destaque dos principais benefícios
- **Footer Completo** - Informações de contato e navegação secundária

## Como Iniciar

### 1. Pré-requisitos

- **Node.js** (versão 16 ou superior)
- **npm**

### 2. Clonar/Baixar o Repositório

- 'git clone https://github.com/Ds007GG/TRABALHO.git'

- 'cd TRABALHO'

### 3. Instalar Dependências

- 'npm install'

### 4. Executar em Desenvolvimento

- 'npm run dev'

A aplicação será iniciada em `http://localhost:5173`.

## Recursos Principais

### Componentes React

- **Header** - Navegação responsiva com links âncora
- **HeroSection** - Banner com gradiente e botão CTA
- **ServiceCard** - Card reutilizável com props (título, descrição, ícone)
- **FeaturesSection** - Showcase de 3 diferenciais principais
- **Footer** - Rodapé com múltiplas colunas de informação

### Responsividade

- Totalmente responsivo para **mobile, tablet e desktop**
- Usa grid system do Bootstrap (`col-12 col-md-6 col-lg-3`)
