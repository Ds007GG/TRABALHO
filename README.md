# NexaFlow

Painel moderno de serviços digitais desenvolvido com React + Vite.

## Tecnologias utilizadas
- React
- Vite
- Tailwind CSS
- Lucide React

## Funcionalidades
- Layout responsivo
- Componentização
- Cards dinâmicos
- Interface moderna

## Integrantes
- Demétrio

## Como executar

npm install

npm run dev
